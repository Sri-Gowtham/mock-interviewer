# AI Mock Interviewer (CrewAI + Streamlit)

This project is a role-based AI mock interviewer built using CrewAI and Streamlit.

Features:
- Role selection (Data Scientist, Web Developer, UI/UX)
- AI-based feedback using CrewAI agents
- Sequential task execution
- Interactive UI

Agents:
- Question Generator
- Feedback Generator

Platform:
- Streamlit Cloud
